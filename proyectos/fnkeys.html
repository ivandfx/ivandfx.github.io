fnKeys

Fix function keys on TKL and compact keyboards under Linux.

Version 3 - July 23, 2025

Only tested on Linux x86-64

Auto install

Copy & paste this into a terminal window:

curl -fsSL https://raw.githubusercontent.com/ivandfx/fnkeys/refs/heads/main/FnKeysAutoInstaller.sh | bash


Manual install

Download Download from GitHub

1. Download fnKeys using the Download button.

2. Go to the folder where fnKeys is located and open a terminal window.

3. Then, copy & paste this:

sudo chmod +x FnKeysFix.sh && ./FnKeysFix.sh

4. Run the installer, and you're done!
