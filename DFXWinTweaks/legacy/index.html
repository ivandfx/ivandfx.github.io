DFX WinTweaks

Hey! How are you in here? This website is no longer maintained.

DFX WinTweaks is a small and versatile tool designed to get system information and tweak your OS beyond what Microsoft allows you. As development progresses, new features will be added.

DFX Tweaker has been renamed to DFX WinTweaks, see why.

Releases in this page are no longer updated. Please go to the new page.

System Requirements

DFX Tweaker Standard: Windows 10, Windows 11.

DFX Tweaker NT6: Windows 7, Windows 8 and Windows 8.1.

How to use

Download any release and run it, no installation required

After that, DFX Tweaker will ask you for Administrator privileges, and then a CMD window will appear with a GUI that you can use by selecting an option with the proper number.

How do release acronyms work?

Releases ending in "H" are just hotfixes.

Hey! Why Windows Defender blocks this script from running?

If Windows Defender flags DFX Tweaker as malware or virus, ignore it and try to run it again.

I'm not really sure, but probably this has something to do with an command that disables Windows Defender, that's why disabling it requires to reboot to Safe Mode.
