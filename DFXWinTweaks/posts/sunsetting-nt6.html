Windows 7, 8 and 8.1 are out of battle - April 7, 2023

Ths post is no longer relevant. Check this other post.


The day has arrived... Anyways, let's be real, everyone knew this would happen soon or later, and here it is, DFX Tweaker no longer supports Windows 7, Windows 8 and Windows 8.1.

Why I did this? Ok, the answer is simple and pretty straightforward. Just think about having to develop a program (or in this case, a script) that relies on the limited features and registry settings that are inside an older operating system that less than 3% of the world use nowadays.

And to make things worse, DFX Tweaker uses Visual Basic Script, (yes, the first day of development I thought this would kill me) a really "basic and old" (don't quote me on that) scripting language so I'm very limited on the things that I can do.

I tried so hard, but developing DFX Tweaker NT6 (using my spare time, that I don't have) is so hard, time-wasting and worthless...

And with that in mind, the NT6 Edition was very weird from a functional standpoint, and I tried to change that releasing a Canary release of NT6, but it's so hard to me to find any feature that Windows 7, 8 or 8.1 doesn't have in handy... I mean these operating systems are pretty well customizable.

I'm really sorry...


This day, DFX Tweaker has lost the battle...

See you again in Windows 10, 11... or maybe 12? Nobody knows :P

Version 2.1 will be the first release to not have an NT6 edition.
